# Frontend and Backend Integration Test

This repository contains an automated test script to verify the integration between the frontend and backend services using Selenium WebDriver and Python.

## Prerequisites

1. **Python**: Ensure Python is installed on your machine. You can download it from [Python's official website](https://www.python.org/downloads/).
2. **Selenium WebDriver**: Install the Selenium WebDriver for Python using pip.
3. **WebDriver**: Download the appropriate WebDriver for your browser (e.g., [ChromeDriver for Chrome](https://sites.google.com/a/chromium.org/chromedriver/)).
